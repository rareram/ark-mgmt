# 인프라 가이드

이 문서는 사내 임직원을 대상으로 인프라 관련 기술, 정책, 사용 절차를 안내하기 위해 작성되었습니다.

## 1. 목적

인프라 가이드의 주요 목적은 다음과 같습니다.

* **클라우드 테스트베드 활용**:
    * **OCI (Oracle Cloud Infrastructure)** 를 중심으로 개발팀의 리소스 테스트, QA팀의 기능 및 성능 검증 등을 지원합니다.
    * 향후 고객사 환경(Oracle DB)과 유사한 클라우드 환경에서 자사 서비스를 운영하기 위한 기술적 기반을 마련합니다.

* **신규 서비스 인프라 및 시스템 활용**:
    * 향후 클라우드 서비스를 하기 위해 파일럿을 운영합니다.

이 가이드를 통해 모든 팀이 표준화된 절차에 따라 인프라를 효율적으로 사용하고, 지속적으로 개선해 나가는 것을 목표로 합니다.

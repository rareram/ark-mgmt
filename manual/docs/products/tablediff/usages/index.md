# 튜토리얼 개요

TableDiff의 기능과 사용법을 **단계별로** 익힐 수 있는 튜토리얼입니다.
처음 사용하시는 분은 **기본 개념과 실행** 단계부터 순서대로 진행해 주세요.

---

## 학습 로드맵

1.  **[기본 개념과 실행](./step1.md)**
    *   TableDiff의 핵심 동작 원리(정렬, 비교, 반영)를 이해하고, 가장 간단한 설정으로 직접 실행해 봅니다.

2.  **[3-Block 설정 구조](./config.md)**
    *   HOCON 설정 파일의 3가지 핵심 블록(Source/Target, Compare, ApplyTo)을 상세히 파헤칩니다.

3.  **[결과 해석과 후처리](./output.md)**
    *   NDJson 결과 파일을 리눅스 도구로 분석하고, Mock 모드를 통해 안전하게 DB에 반영하는 방법을 배웁니다.

4.  **[실패로 배우는 제약조건](./troubleshooting.md)**
    *   자주 발생하는 5가지 실패 사례를 직접 재현해보며, 반드시 지켜야 할 제약 조건들을 체득합니다.

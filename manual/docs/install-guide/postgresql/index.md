# 개요

이 가이드는 PostgreSQL 데이터베이스를 설치하고, `TableDiff` 및 CDC 기능 테스트에 필요한 환경을 구성하는 방법을 안내합니다.

설치는 다음 두 단계로 진행됩니다.

1. **[PostgreSQL 설치](./postgresql-install.md)**: Docker를 활용하여 PostgreSQL을 빠르고 간편하게 설치합니다.
2. **[데이터베이스 및 테이블 생성](./database-setup.md)**: 설치된 PostgreSQL에 접속하여 테스트용 데이터베이스를 생성하고, 필요한 테이블 및 초기 데이터를 구성합니다.

각 문서를 순서대로 따라 진행하는 것을 권장합니다.
